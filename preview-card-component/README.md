# Stats Preview Card Component

This project is a solution to the Frontend Mentor **Stats Preview Card Component** challenge. It demonstrates responsive design and focuses on creating a visually appealing card component with modern CSS and semantic HTML.

## Overview

The **Stats Preview Card Component** is a simple card design that displays information about a business, such as insights and statistics. It adapts to various screen sizes using CSS media queries to ensure a great user experience on both desktop and mobile devices.

### Features

- **Fully Responsive**: Works seamlessly across desktop and mobile devices.
- **Custom Fonts**: Uses Google Fonts (`Inter` and `Lexend Deca`).
- **Modern Design**: Includes a visually appealing overlay effect on the image.

---

## Technologies Used

- HTML5: For semantic structure.
- CSS3: For styling and responsive design.
- [Frontend Mentor Starter Files](https://www.frontendmentor.io/): For the initial assets and challenge guidelines.

## Setup

1. Clone the repository:
   ```bash
   git clone <repository_url>
   ```
2. Navigate to the project directory:
   ```bash
   cd stats-preview-card-component
   ```
3. Open the `index.html` file in your browser to view the project.

---

## Responsive Design

This project is designed to adapt to different screen sizes:
- **Desktop View**: The card displays the image and text side by side.
- **Mobile View**: The layout switches to a vertical design with the image on top.
